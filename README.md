# Mental Health Prediction and Global Disorder Analysis

This project explores **global mental health trends** using multiple datasets and applies **machine learning models** to predict mental health indicators.  
It includes **statistical analysis**, **interactive Plotly visualizations**, **dynamic subplots**, **correlation studies**, and **regression models** to understand mental health patterns.

---

##  Datasets Used

1. `mental-illnesses-prevalence.csv`
2. `adult-population-covered-in-primary-data-on-the-prevalence-of-mental-illnesses.csv`
3. `depressive-symptoms-across-us-population.csv`
4. `number-of-countries-with-primary-data-on-prevalence-of-mental-illnesses-in-the-global-burden-of-disease-study.csv`

These datasets contain global, regional, and U.S. mental health statistics.
